# Calculator
A simple Python-based calculator that performs basic arithmetic operations like addition, subtraction, multiplication, division, square, cube and roots using a clean command-line interface.

## 🚀 Features

- Addition 
- Subtraction 
- Multiplication
- Division 
- Square
- Cube
- Square root
- Continuous calculations without restarting the program
- Input validation for safer execution

  ## Programming language used:
                 PYTHON3
